# DeepSeek Assistant – Feature Showcase

Here are 5 key features of the extension, each with a screenshot from the panel interface.

---

## 1. 🔍 Search – History & Current Conversation

- **Dual-mode search** : left sidebar history + current page content  
- Keyword highlighting, chronological sorting, CSV export  
- Copy search results as plain text or Markdown

![Search Panel](./screenshots/search-tab.png)

---

## 2. ⭐ Favorites – Conversations & Messages

- Favorite entire conversations or single messages (`Ctrl+Shift+X`)  
- Folder management (create, rename, delete, search)  
- Add notes to message favorites, jump back with exact positioning  
- Today/Week statistics, JSON import/export

![Favorites Panel](./screenshots/favorites-tab.png)

---

## 3. 🎨 Appearance – Themes & Panel Control

- 6 preset themes + custom color picker + follow system theme  
- Drag & drop panel (position remembered per conversation)  
- Preset panel sizes (original/small/medium/large) via floating button long-press  
- Minimize to floating button, hotkey `Alt+K` to open

![Appearance Panel](./screenshots/appearance-tab.png)

---

## 4. 🌐 Internationalization – English / Chinese

- Full UI translation (≈120 keys) – tabs, buttons, tooltips  
- Auto-detects browser language, switch instantly without refresh  
- Easily extensible for new languages

![I18n Panel](./screenshots/i18n-tab.png)

---

## 5. 🧩 Others – Storage Monitor, Settings & More

- Real-time storage usage (warning at 80% / 95%)  
- One-click reset panel position & size  
- Export filenames with second-precision timestamps  
- Sidebar collapse hint, boundary protection, font rendering optimizations

![Others / Settings Panel](./screenshots/others-tab.png)

---

> All screenshots are taken from the DeepSeek Assistant panel in Microsoft Edge.  
> Actual interface may vary slightly with theme and language settings.
