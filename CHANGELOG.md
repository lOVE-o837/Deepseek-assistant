# 更新日志

🎉 DeepSeek 助手插件更新啦！v1.1.1 来了～

这次更新聚焦「收藏管理」和「搜索效率」，让你用得更顺手。

✨ 新功能速览

1️⃣ 收藏夹统计条
   对话收藏 + 消息收藏，数字实时显示。切换文件夹后自动刷新，文件夹里存了多少东西，一眼看清。

2️⃣ 消息收藏备注
   每条收藏的消息都能添加备注，支持编辑、删除、搜索、高亮。重要的知识点、待办事项，直接写在备注里，查找更方便。

3️⃣ 当前对话结果复制
   在当前对话中搜索后，点击「复制结果」按钮，一键带走所有匹配项（标题+片段+链接），整理笔记超省心。

4️⃣ 面板打开自动聚焦
   打开面板后，光标自动落在搜索框，不用再点一下输入框就能直接打字。

🔧 优化 & 修复
- 搜索范围提示更清晰：受页面虚拟滚动限制，提示滚动后重试或使用官方全局搜索。
- 导出/导入完整保留备注字段，备份不丢任何数据。
- 移动文件夹后统计数字即时更新。
- 修复消息收藏删除后统计不刷新、标签页切换聚焦失效等 bug。

## [v1.0.0] - 2026-05-10

### 🎉 首个正式版本发布

**核心功能**：
- 🔍 **双模式搜索**：历史对话列表检索 + 当前页面内容搜索，支持时间排序与关键词红底白字高亮
- ⭐ **双重收藏系统**：对话收藏（悬浮按钮/快捷键/右键菜单）+ 消息收藏（逐条收藏提问与回答）
- 💬 **消息精准跳转**：收藏单条消息后，点击即可跳转回原对话并自动滚动定位
- 🎨 **增强主题系统**：6套预设主题 + 自定义取色器 + 跟随系统深色/浅色模式自动切换
- ⌨️ **高效快捷键**：`Alt+K` 呼出面板，`Ctrl+Shift+X` 一键收藏
- 📤 **数据管理**：支持 JSON 和 TXT 格式的对话收藏导出备份与导入恢复
- 📝 **收藏备注**：收藏时可添加自定义备注，搜索框支持备注关键词检索
