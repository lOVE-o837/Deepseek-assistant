# 更新日志 (CHANGELOG)

所有值得关注的变更都会记录在此文件中。格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [1.4.1] - 2026-06-06

### ✨ 新增
- **用户反馈渠道**：首次打开面板的友好提示及设置页固定入口，帮助项目收集使用需求。
- **独家主题特效**：新增隐藏款“深海极客”主题，包含动态光晕、流光边框与粒子背景，可通过特定互动方式解锁。
- 设置页新增“专属奖励”兑换功能，配合社区活动使用。
- 设置页收藏统计卡片动态显示当前文件夹名称。

### 🔧 优化
- 初次安装或重置后，面板立即呈现默认深海蓝主题，不再短暂显示白色。
- 浅色主题下（深海蓝、极简白、护眼绿、暖光橙）导航栏文字更清晰。
- 面板顶部空白区域移除，界面更紧凑。
- 隐藏主题特效仅在该主题下显示，切换后自动消失。

### 🐛 修复
- 修复删除收藏后悬浮按钮未实时恢复默认状态的问题。
- 修复切换回其他主题后，深色背景或特效残留的问题。
- 修复隐藏主题下文字模糊、导航栏文字不可见等问题。
- 修复粒子特效在隐藏主题下丢失的显示错误。
- 修复主题切换后面板出现多余滚动条的问题。
- 清理了重复的导航栏样式规则，提升可维护性。

## [1.4.0] - 2026-06-04

### 🏗️ 架构重构
- **Content Script 模块化拆分**：将约 2000 行 `content.js` 拆分为 7 个独立模块（utils, panel, bookmark, float-btn, sidebar, search, core），通过 `window.__DS__` 命名空间通信，各模块可独立维护。

### ✨ 优化
- **样式统一**：「移动到文件夹」按钮改为绿色风格，与打开/备注（蓝色）、删除（红色）保持一致，hover 均有颜色反转反馈。
- **统计增强**：设置页收藏统计卡片动态显示当前选中的文件夹名称，随文件夹切换实时更新。
- **Content Script 国际化**：长按菜单、收藏弹窗、消息提示等页面端 UI 全部支持中英文切换（使用 `lang` 变量 + 三元表达式，绕过面板 `t()` 函数上下文隔离限制）。
- **空状态显示修复**：收藏夹和消息收藏空状态改用三元表达式直接判断语言，彻底解决 `I18N` 初始化时序导致的显示键名问题（`bookmarksEmpty` 等）。
- **Markdown 导出增强**：导出时完整保留 AI 代码块（自动识别语言并添加 Markdown 代码围栏）。
- **语言切换同步**：`content-core.js` 中 `lang` 变量随 `settings-changed` 消息同步更新，确保页面端 UI 实时切换语言。
- **字体渲染优化**：Windows ClearType 原生渲染，彻底解决笔画重叠黑块问题。
- **存储百分比精确化**：小数点后两位精确显示。
- **Toast 提示完善**：语言切换成功提示、收藏/删除/定位/超时等 20+ 处 Toast 国际化遗漏全部补全。
- **备注编辑弹窗国际化**：`prompt()` 弹窗根据当前语言显示中英文提示。

### 🐛 修复
- 修复删除收藏后右下角悬浮按钮无法即时恢复默认外观的问题，现在按钮状态实时同步。
- 修复恢复默认设置按钮无反应（`computeThemeCSSVars` 遗漏迁移到 `module-theme.js`）。
- 修复当前对话搜索角色标签（AI回复/用户消息）语言切换不更新（`content.js` 硬编码源头修复）。
- 修复搜索排序按钮初次显示中文、切换后不更新。
- 修复收藏夹卡片按钮、操作菜单语言切换后不刷新。
- 修复导出弹窗、取色器弹窗中文未国际化。
- 修复相对时间显示（刚刚/分钟前/小时前/天前）未国际化。
- 修复英文 `messagesCountSuffix` 显示为键名本身。
- 修复设置页产品名和版本号丢失。
- 修复 `panel.html` 中空状态提示文字未添加 `data-i18n` 属性。
- 修复移动到文件夹弹窗匹配计数和"全部（无分类）"按钮未国际化。
- 修复收藏夹搜索计数条"共 X 项，匹配 Y 项"未国际化。
- 修复取色器弹窗标题双重图标问题。
- 修复隐私声明和安全提醒点击变半透明模糊。

## [1.3.0] - 2026-05-28

### 🏗️ 架构重构
- **面板代码模块化**：将原 `panel.js`（2140 行）拆分为 11 个独立模块，各司其职，极大降低维护难度。

### 🌐 国际化
- **全界面中英文切换**：覆盖 6 个标签页，约 120 个翻译键，语言切换即时生效，静态文本通过 `data-i18n` 统一刷新。
- 不支持的语言（俄语/印尼语等）提示"敬请期待"。

### 💾 存储监控
- 设置页新增存储用量卡片，实时显示百分比，超过 80% 橙色警告，超过 95% 红色警告。

### 🎨 UI 优化
- 关于页两列布局，字体渲染优化，空状态文字加亮，返回按钮深色模式可见。

### 🐛 修复
- 修复恢复默认设置按钮无反应、搜索排序按钮语言切换不更新、多个国际化遗漏等问题共十余项。

## [1.0.0] - 2026-05-10

### 🎉 首个正式版本
- **双模式搜索**：历史对话检索 + 当前页面内容搜索，时间排序与红底白字高亮。
- **双重收藏**：对话收藏（悬浮按钮/快捷键/右键）与消息收藏（单条提问/回答）。
- **消息跳转**：收藏消息点击跳转回原对话并精准定位。
- **6 套主题** + 自定义取色器 + 跟随系统深色模式。
- **快捷键**：Alt+K 呼出面板，Ctrl+Shift+X 收藏。
- **数据管理**：JSON / TXT 导出备份与导入恢复。
- **收藏备注**：可添加自定义备注，搜索支持检索备注。
