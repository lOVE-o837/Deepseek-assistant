# 更新日志

🎉 DeepSeek 助手插件更新啦！v1.1.1 来了～

这次更新聚焦「收藏管理」和「搜索效率」，让你用得更顺手。

✨ 新功能速览

1️⃣ 收藏夹统计条
   对话收藏 + 消息收藏，数字实时显示。切换文件夹后自动刷新，文件夹里存了多少东西，一眼看清。

2️⃣ 消息收藏备注
   每条收藏的消息都能添加备注，支持编辑、删除、搜索、高亮。重要的知识点、待办事项，直接写在备注里，查找更方便。

3️⃣ 当前对话结果复制
   在当前对话中搜索后，点击「复制结果」按钮，一键带走所有匹配项（标题+片段+链接），整理笔记超省心。

4️⃣ 面板打开自动聚焦
   打开面板后，光标自动落在搜索框，不用再点一下输入框就能直接打字。

🔧 优化 & 修复
- 搜索范围提示更清晰：受页面虚拟滚动限制，提示滚动后重试或使用官方全局搜索。
- 导出/导入完整保留备注字段，备份不丢任何数据。
- 移动文件夹后统计数字即时更新。
- 修复消息收藏删除后统计不刷新、标签页切换聚焦失效等 bug。

# v1.3.0 Release Notes

## 🏗️ 架构重构：代码模块化拆分

将原 `panel.js`（2140 行单文件）拆分为 **11 个独立模块**，按职责清晰分离：

| 模块 | 职责 |
|:---|:---|
| `module-i18n.js` | 语言包、翻译函数、语言切换、静态文本刷新 |
| `module-storage.js` | 存储读写、安全通信、就绪握手 |
| `module-dom.js` | 纯 DOM 工具函数（Toast / 格式化 / 防抖 / 高亮） |
| `module-theme.js` | 主题定义、应用、取色器、颜色计算工具 |
| `module-panel.js` | 标签切换、面板控制、设置子视图、存储用量 |
| `module-history.js` | 历史搜索、CSV 导出、工具栏刷新 |
| `module-search.js` | 当前对话搜索、结果复制 |
| `module-bookmarks.js` | 对话收藏、导出/导入 |
| `module-messages.js` | 消息收藏、卡片事件 |
| `module-folders.js` | 文件夹管理、移动弹窗、统计条 |
| `panel.js` | 主控：全局状态、DOM 缓存、初始化、事件绑定 |

**收益**：各模块独立维护，新增功能只需改对应文件，不再触碰 2000+ 行巨型文件。
