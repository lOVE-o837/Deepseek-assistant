# KNOWN_ISSUES

## 已知问题与限制

### 当前对话搜索范围受限

**现象**：在当前对话页面中搜索时，只能匹配视口附近约 5-8 条消息，远处的消息不会被搜到。

**原因**：DeepSeek 使用虚拟滚动技术渲染消息列表，DOM 中仅保留可见区域附近的少量消息元素。扩展的搜索功能依赖 DOM 查询，无法触达已被虚拟滚动卸载的消息。

**影响**：搜索结果可能不完整，需手动滚动页面加载更多消息后重新搜索。

**计划**：后续版本将通过主动触发虚拟滚动加载机制来扩展搜索范围（规划中 v1.5.0 高级版功能）。

---

### 消息收藏跳转定位可能超时

**现象**：从收藏夹点击消息跳转回原对话时，偶尔提示「未找到该消息」或「消息未加载」。

**原因**：同样受虚拟滚动影响，目标消息可能不在当前 DOM 中。扩展会尝试自动向上滚动触发加载（最多轮询 25 次），但极远处消息可能超出轮询时长。

**临时方案**：收到超时提示后，手动向上滚动对话页面加载更早的消息，然后重新点击跳转。

---

### 侧边栏收起时历史对话抓取受限

**现象**：面板历史搜索抓取的对话数量明显偏少（仅几条）。

**原因**：扩展通过查询 DeepSeek 左侧侧边栏的 DOM 获取对话列表。当侧边栏被收起或处于狭窄模式时，对话列表项可能未被完整渲染。

**影响**：搜索结果少于 3 条时，面板会提示「展开侧边栏获取更多记录」。

---

### DOM 结构变化风险

**现象**：部分功能（搜索定位、消息角色识别、侧边栏抓取）依赖 DeepSeek 页面的 CSS 类名和 DOM 结构。

**原因**：DeepSeek 平台更新可能修改前端类名（如 `._7780f2e`、`.ds-message` 等），导致选择器失效。

**应对**：扩展使用多策略兜底查询，即使主选择器失效仍会回退到备选方案。如完全失效，请提交 Issue 并附上 DeepSeek 页面版本信息。

---

### 中英文导出文件跨语言导入兼容性

**现象**：中文环境下导出的 JSON 备份，在英文环境下导入时可能出现类型识别异常。

**原因**：导出文件中的角色标识（AI/用户）、时间格式等字段在不同语言下的显示文本不同，导入解析逻辑依赖当前语言环境。

**建议**：导入备份时使用与导出时相同的语言设置。
