# CODE_OF_CONDUCT

## 行为准则

### 我们的承诺

为了营造一个开放和友好的社区环境，作为本项目的贡献者和维护者，我们承诺：无论年龄、体型、残障、种族、性别认同与表达、经验水平、国籍、个人外貌、宗教、性取向或经济地位如何，参与本项目不会受到骚扰。

### 我们的准则

有助于创造积极环境的行为包括：

- 使用友好和包容的语言
- 尊重不同的观点和经验
- 虚心接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员展现同理心

不可接受的行为包括：

- 发布性暗示的言语或图像，以及不受欢迎的性关注
- 挑衅、侮辱/贬低性评论，以及人身或政治攻击
- 公开或私下骚扰
- 未经明确许可，发布他人的私人信息（如物理地址或电子邮箱）
- 在专业场合被合理认定为不适当的其他行为

### 我们的责任

项目维护者有责任阐明可接受行为的标准，并对任何不可接受的行为采取适当且公正的纠正措施。

项目维护者有权利和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、Wiki 编辑、Issue 和其他贡献，并暂时或永久禁止任何他们认为有不当、威胁、冒犯或有害行为的贡献者。

### 适用范围

本行为准则适用于项目空间内以及个人代表本项目或其社区时的公共场合。

### 执行

如遇到辱骂、骚扰或其他不可接受的行为，可通过在 GitHub Issues 中标记 @{{你的用户名}} 进行举报。所有投诉将被审查和调查，并根据情况作出必要和适当的回应。项目团队有义务对事件举报者保密。

### 归属

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org) 1.4 版。
