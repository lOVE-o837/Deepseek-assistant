# Contributing to DeepSeek 助手

感谢你的关注！欢迎通过以下方式参与贡献：

## 报告 Bug
如果你发现任何问题，请在 [Issues](https://github.com/IOVE-o837/DeepSeek-Assistant/issues) 页面提交，并包含：
- 问题现象描述
- 复现步骤
- 浏览器版本与插件版本
- 相关截图或控制台错误信息

## 功能建议
如果你有新功能的想法，欢迎在 [Discussions](https://github.com/IOVE-o837/DeepSeek-Assistant/discussions) 中提出。

## Pull Request 流程
1. Fork 本仓库
2. 创建你的功能分支 (`git checkout -b feature/your-feature`)
3. 提交你的修改 (`git commit -m 'Add some feature'`)
4. 推送到分支 (`git push origin feature/your-feature`)
5. 创建一个 Pull Request

## 行为准则
本项目遵守 [Contributor Covenant](https://www.contributor-covenant.org/) 行为准则。请保持友善、尊重他人的贡献。
