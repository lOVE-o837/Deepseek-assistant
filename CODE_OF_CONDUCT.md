# 行为准则

## 我们的承诺

为营造一个开放、友好的环境，我们作为贡献者和维护者，承诺让参与本项目不受年龄、体型、残疾、种族、性别认同与表达、经验水平、国籍、个人外貌、宗教信仰、性取向等任何形式的骚扰。

## 我们的准则

有助于创造积极环境的行为包括：
- 使用友好、包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表现出同理心

不可接受的行为包括：
- 发布他人的私人信息，如物理地址或电子邮箱
- 使用与性相关的语言或图像，以及不受欢迎的性关注或性骚扰、恶意挑衅、侮辱性评论
- 其他被合理认为不适合在专业场合中的行为

## 我们的责任

项目维护者有责任明确可接受行为的标准，并对任何不可接受的行为采取适当、公正的纠正措施。维护者有权删除、编辑或拒绝与本行为准则不一致的评论、提交、代码、问题和其他贡献，或暂时、永久禁止任何贡献者从事其他被认为不当、具有威胁性、冒犯性或有害的行为。

## 适用范围

本行为准则适用于项目的所有空间，也适用于个人代表项目或其社区的公共场所。代表项目或社区的行为包括使用官方项目邮箱、通过官方社交媒体账户发帖，或在线上或线下活动中担任指定代表。

## 报告问题

如果你遇到或目睹了违反本行为准则的情况，请在 GitHub Issues 中联系我们，或通过 Edge 商店页面提供的邮箱联系项目维护者。

## 归属

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/)，版本 2.0。
